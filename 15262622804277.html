弹性布局

前言

在传统的布局方案中，我们主要是用display、float、position等属性，它们已经帮助我们解决了很多布局问题。

flex布局方案于2009年被W3C提出，短短几行代码，就能实现各种页面布局效果。在caniuse中查询，可以看到目前所有现代浏览器都能完美支持该属性。

概念

Flex 是 Flexible Box 的缩写，意为"弹性布局"，主要思想是给予容器控制内部元素高度和宽度的能力。它为盒状模型提供极大的灵活性。

定义容器：

.box{
    display: -webkit-box; /* 老版本语法: Safari, iOS, Android browser, older WebKit browsers. */
    display: -moz-box; /* 老版本语法: Firefox (buggy) */
    display: -ms-flexbox; /* 混合版本语法: IE 10 */
    display: -webkit-flex; /* 新版本语法: Chrome 21+ */
    display: flex; /* 新版本语法: Opera 12.1, Firefox 22+ */ 
}

使用flex布局的案例中，设置了flex属性的父元素一般称为Flex 容器（flex container），简称"容器"。
它的所有子元素自动成为容器成员，称为 Flex 项目（flex item），简称"项目"。

属性列表

容器属性详解

1、flex-direction

决定主轴的方向，即项目排列方向。一共有4个值：

  • row：主轴为水平方向，项目沿主轴从左至右排列
  • column：主轴为竖直方向，项目沿主轴从上至下排列
  • row-reverse：主轴水平，项目从右至左排列，与row反向
  • column-reverse：主轴竖直，项目从下至上排列，与column反向

示例：


2、flex-wrap

默认情况下，item排列在一条线上，即主轴上，flex-wrap决定当排列不下时是否换行以及换行的方式。一共有3个值：

  • nowrap：自动缩小项目，不换行
  • wrap：换行，且第一行在上方
  • wrap-reverse：换行，第一行在下面

示例：



3、flex-flow

是flex-direction和flex-wrap的简写形式。默认值为row nowrap，即横向排列不换行。

4、justify-content

决定item在主轴上的对齐方式。一共有5个值：

  • flex-start：左对齐
  • flex-end：右对齐
  • center：居中对齐
  • space- between：两端对齐
  • space-around：沿轴线均匀分布

示例：

5、align-items

决定了item在交叉轴上的对齐方式。一共有5个值、，在主轴水平的情况下：

  • flex-start：顶端对齐
  • flex-end：底部对齐
  • center：竖直方向上居中对齐
  • baseline：item第一行文字的底部对齐
  • stretch：当item未设置高度时，item将和容器等高对齐

示例：

6、align-content

该属性定义了当有多根主轴时，即item不止一行时，多行在交叉轴轴上的对齐方式。
注意当有多行时，定义了align-content后，align-items属性将失效。一共有6个值，在主轴水平的情况下：

  • flex-start：左对齐
  • flex-end：右对齐
  • center：居中对齐
  • space-between：两端对齐
  • space-around：沿轴线均匀分布
  • stretch：各行将根据其flex-grow值伸展以充分占据剩余空间

flex item 项目属性详解

1、order

order的值是整数，默认为0，整数越小，item排列越靠前。

示例：

<div class="flex">
    <div class="item" style="order:4"><p>item 1</p></div>
    <div class="item" style="order:3"><p>item 2</p></div>
    <div class="item" style="order:2"><p>item 3</p></div>
    <div class="item" style="order:1"><p>item 4</p></div>
</div>

效果如图：

2、flex-grow

定义了当flex容器有多余空间时，item是否放大。
默认值为0，即当有多余空间时也不放大；
可能的值为整数，表示不同item的放大比例，即当有多余空间时item1、item2、和item3以1：2:3的比例放大。

<div class="flex">
    <div class="item1" style="flex-grow:1"><p>item 1</p></div>
    <div class="item2" style="flex-grow:2"><p>item 2</p></div>
    <div class="item3" style="flex-grow:3"><p>item 3</p></div>
</div>

3、flex-shrink

定义了当容器空间不足时，item是否缩小。
默认值为1，表示当空间不足时，item自动缩小。
其可能的值为整数，表示不同item的缩小比例，负值对该属性无效。

4、flex-basis

表示项目在主轴上占据的空间，默认值为auto。

<div class="flex">
    <div class="item" style="flex-basis:40px"><p>item 1</p></div>
    <div class="item" style="flex-basis:80px"><p>item 2</p></div>
    <div class="item" style="flex-basis:160px"><p>item 3</p></div>
</div>

效果如图：

5、flex

flex属性是flex-grow、flex-shrink和flex-basis三属性的简写总和，默认值为0 1 auto，后两个属性可选。也是我们在CSS中最常使用的写法。

6、align-self

align-self属性允许item有自己独特的在交叉轴上的对齐方式，可覆盖align-items属性。
它一共有6个值，默认值为auto。

  • auto：和父元素align-self的值一致
  • flex-start：顶端对齐
  • flex-end：底部对齐
  • center：竖直方向上居中对齐
  • baseline：item第一行文字的底部对齐
  • stretch：当item未设置高度时，item将和容器等高对齐
